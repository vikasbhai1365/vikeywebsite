1. Write a program to evaluate simple interest of a given principle, rate and time.


Program :

import java.util.Scanner;
class SimInter
{
public static void main(String args[])
{
int amount,n;
float rate,si;
Scanner s=new Scanner(System.in);
System.out.println("Enter principal amount :");
amount=s.nextInt();
System.out.println("Enter the rate of interest :");
rate=s.nextFloat();
System.out.println("Enter Time :");
n=s.nextInt();
si=(amount*rate*n)/100;
System.out.println("Simple Interest is : "+si);
}
}

2. A motor cycle dealer sells two-wheelers to his customer on loan, which is to be repaid in 5 years. The dealer charges simple interest for the whole term on the day of giving the loan itself. The total amount is then divided by 60(months) and is collected as equated monthly instalment (EMI). Write a program to calculate the EMI for a loan of Rs. X, where X is given from command line argument. Print the EMI value in rupees.


Program :

import java.util.Scanner;
class FindEmi
{
public static void main(String args[])
{
int n;
float rate;
double amount,si,emi;
amount=Integer.parseInt(args[0]);
rate=Float.parseFloat(args[1]);
n=60;
si=(amount*rate*n)/100;
amount=amount+si;
emi=amount/n;
System.out.println("Simple interest is :"+si);
System.out.println("Total amount with interest is :"+amount);
System.out.println("Emi is :"+emi);
}
}

3. A car accessories shop assigns code 1 to seat covers, 2 to steering wheel covers , 3 to car lighting and 4 for air purifiers. All other items have code 5 or more. While selling the goods, a sales tax of 2% to seat covers ,3% to steering wheel covers, 4% to car lighting, 2.5% to air purifiers and 1.2% for all other items is charged. A list containing the product code and price is given for making a bill. Write a java program using switch statements to prepare a bill.


Program :

import java.util.Scanner;
class Shop
{
public static void main(String args[])
{

productCode=s.nextInt();
System.out.println("Enter price :");
price=s.nextFloat();
switch(productCode)
{
case 1:
taxRate=2.0d;
break;
case 2:
taxRate=3.0d;
break;
case 3:
taxRate=4.0d;
break;
case 4:
taxRate=2.5d;
break;

taxRate=1.2d;

4. Write a java program to scan 3 integer values from the command line argument and display the maximum number using conditional operator.


Program :

class FindMax
{
public static void main(String args[])
{
int a,b,c,max;
a=Integer.parseInt(args[0]);
b=Integer.parseInt(args[1]);
c=Integer.parseInt(args[2]);
max=(a>b)?((a>c)?a:c):((b>c)?b:c);
System.out.println("Max is :"+max);
}
}

5. Write a program to calculate the hypotenuse of right angled triangle when other sides of the triangle are given. (Hypotenuse = square root (x*x + Y *Y))


Program :

{

{

7. Create a complex number class. The class should have a constructor and methods to add, subtract and multiply two complex numbers and to return the real and imaginary parts.


Program :

class Complex { int real; int image; Complex() { } Complex(int a,int b) { real=a; image=b; } Complex add(Complex c1,Complex c2) { Complex sum=new Complex(); sum.real=c1.real+c2.real; sum.image=c1.image+c2.image; return sum; } Complex sub(Complex c1,Complex c2) { Complex sub=new Complex(); sub.real=c1.real-c2.real; sub.image=c1.image-c2.image; return sub; } Complex mul(Complex c1,Complex c2) { Complex mul=new Complex(); mul.real=c1.real*c2.real + c1.image*c2.image; mul.image=c1.real*c2.real - c1.image*c2.image; return mul; } } class MainComplex { public static void main(String args[]) { Complex c1=new Complex(10,20); Complex c2=new Complex(5,6); Complex c3=new Complex(); c3=c3.add(c1,c2); System.out.println("Sum :"+c3.real+"i"+c3.image); c3=c3.sub(c1,c2); System.out.println("Sub :"+c3.real+"i"+c3.image); c3=c3.mul(c1,c2); System.out.println("mul :"+c3.real+"i"+c3.image); } }

9. A bank gives 6.5% per annum interest on deposits made in that bank. Write a program to calculate the total amount that a person will receive after the end of 5 years for a deposit of Rs.5000 for compound interest. Create necessary methods and constructors too.


Program :

import java.util.Scanner; import java.lang.Math; class CompoundInterest { int deposit,time; double rate,amount; CompoundInterest(int d,double r,int t) { deposit=d; rate=r; time=t; } double calc() { amount=deposit*Math.pow((1+rate/100),time); return amount; } public static void main(String args[]) { int deposit=5000,time=5; double rate=6.5; double cInterest,totalAmount; CompoundInterest c1=new CompoundInterest(deposit,rate,time); totalAmount=c1.calc(); cInterest=totalAmount-deposit; System.out.println("_______________________________"); System.out.println("Deposit:"+deposit); System.out.println("Rate of Interest :"+rate); System.out.println("Compound Interest is :"+cInterest); System.out.println("_______________________________"); } }
