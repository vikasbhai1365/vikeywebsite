4. Write a program that accepts a string from command line and perform following operations: 1. Display each character on separate line in reverse order. 2. Count total number of chracters and display each character's position too. 3. Identify that whether the string is palindrom or not. 4. Count total number of uppercase and lowercase characters in it.


Program :

import java.util.Scanner; class StringOperations { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Enter a string: "); String mystring = sc.next(); // 1. Display each character on a separate line in reverse order int n=mystring.length(); System.out.println("Characters in reverse order:"); for (int i = n-1; i >= 0; i--) { System.out.println(mystring.charAt(i)); } // 2. Count total number of characters and display each character's position System.out.println("\nTotal number of characters: " +n); System.out.println("Character positions:"); for (int i = 0; i < n; i++) { System.out.println(mystring.charAt(i) + ": " + (i + 1)); } // 3. Identify whether the string is a palindrome or not boolean isPalindrome = true; for (int i = 0; i < n / 2; i++) { if (mystring.charAt(i) != mystring.charAt(n - 1 - i)) { isPalindrome = false; break; } } if (isPalindrome) { System.out.println("\nThe string is a palindrome."); } else { System.out.println("\nThe string is not a palindrome."); } // 4. Count total number of uppercase and lowercase characters int uppercaseCount = 0, lowercaseCount = 0; for (int i = 0; i < n; i++) { char ch = mystring.charAt(i); if (Character.isUpperCase(ch)) { uppercaseCount++; } else if (Character.isLowerCase(ch)) { lowercaseCount++; } } System.out.println("\nUppercase characters count: " + uppercaseCount); System.out.println("Lowercase characters count: " + lowercaseCount); } }

5. Write a Java program to input n integer numbers and display lowest and second lowest number. Also handle the different exceptions possible to be thrown during execution.


Program :

import java.util.Scanner; import java.util.Arrays; import java.util.InputMismatchException; class LowestNumbers { public static void main(String[] args) { Scanner sc = new Scanner(System.in); int n; int[] numbers; try { System.out.print("How many numbers you want to enter : "); n = sc.nextInt(); if (n <= 1) { System.out.println("Please enter a valid positive integer greater than 1."); return; } numbers = new int[n]; System.out.println("Enter " + n + " integer numbers:"); for (int i = 0; i < n; i++) { numbers[i] = sc.nextInt(); } Arrays.sort(numbers); int lowest = numbers[0]; int secondLowest=numbers[1]; System.out.println("Lowest number: " + lowest); System.out.println("Second lowest number: " + secondLowest); } catch (InputMismatchException e) { System.out.println("Input should be an integer."); } catch (Exception e) { System.out.println("Some Error"); } } }

7.Write an application that accepts marks of three different subject from user. Marks should be between 0 to 100, if marks of any of the subject is not belong to this range, generate custom exception out of RangeException. If marks of each subjects are greater than or equal to 40 then display message “PASS” along with percentage, otherwise display message “FAIL”. Also write exception handling code to catch all the possible runtime exceptions likely to be generated in the program.


Program :

import java.util.*; class RangeException extends Exception { RangeException(String message) { super(message); } } class Main { public static void main(String args[]) { Scanner sc=new Scanner(System.in); int marks1=0,marks2=0,marks3=0; try{ System.out.println("Enter marks of three subjects :"); marks1=sc.nextInt(); marks2=sc.nextInt(); marks3=sc.nextInt(); if((marks1 > 100 || marks1 < 0)||(marks2 > 100 || marks2 < 0)||(marks3 > 100 || marks3 < 0)) { throw new RangeException("Enter marks between(0-100)."); } else if((marks1 > 40)&&(marks2 > 40)&&(marks3 > 40)) { System.out.println("Congratulations , You are Passed"); } else { System.out.println("Sorry , You are Failed"); } int totalmarks=marks1+marks2+marks3; System.out.println("totalmarks :"+totalmarks); double per=(double)totalmarks/3; System.out.println("Percentage :"+per+"%"); } catch(InputMismatchException e) { System.out.println("Here ,Input mismatch occured."); } catch(RangeException e) { System.out.println(e.getMessage()); } catch(Exception e) { System.out.println("Some Error."); } } }

8.Write a program which takes the age of 5 persons from command line and find the average age of all persons. The program should handle exception if the argument is not correctly formatted and custom exception if the age is not between 1 to 100.


Program :

class RangeException extends Exception { RangeException(String Message) { super(Message); } } class Main { public static void main(String args[]) { int ages[]=new int[5]; int avg=0,totalage=0; try { for(int i=0;i<5;i++) { ages[i]=Integer.parseInt(args[i]); if(ages[i] < 0 || ages[i] > 100) { throw new RangeException("Enter age between (0-100)."); } totalage=totalage + ages[i]; } avg=totalage/5; System.out.println("Average of age is :"+avg); } catch(ArrayIndexOutOfBoundsException e) { System.out.println(" Array index is out of bound."); } catch(RangeException e) { System.out.println(e.getMessage()); } catch(Exception e) { System.out.println("Some Error."); } } }
