4.Write a program to sort the array elements using bubble sort technique.


Program :

import array as arr myarray=arr.array('i',[4,2,3,1]) n=len(myarray) print("Before sorting :",myarray) for i in range(n-1): for j in range(0,n-i-1): if(myarray[j]>myarray[j+1]): myarray[j],myarray[j+1]=myarray[j+1],myarray[j] print("After Sorting :",myarray)

13.Create a program name “employee.py” and implement the functions DA, HRA, PF, and ITAX. Create another program that uses the function of employee module and calculates gross and net salaries of an employee.


Program :

employee.py file

def DA(basic_salary): #10% of the basic salary return basic_salary*0.10 def HRA(basic_salary): # 15% of the basic salary return basic_salary*0.15 def PF(basic_salary): #12% of the basic salary return basic_salary*0.12 def ITAX(gross_salary): #10% of gross salary if gross_salary>50000: return gross_salary*0.10 else: return 0.0

main.py file

import employee def calculate_gross_salary(basic_salary): da=employee.DA(basic_salary) hra=employee.HRA(basic_salary) gross_salary=basic_salary+da+hra return gross_salary def calculate_net_salary(basic_salary): pf=employee.PF(basic_salary) gross_salary=calculate_gross_salary(basic_salary) itax=employee.ITAX(gross_salary) net_salary=gross_salary-(pf+itax) return net_salary #here we will call the functions basic_salary=float(input("Enter Basic Salary :")) gross_salary=calculate_gross_salary(basic_salary) net_salary=calculate_net_salary(basic_salary) print("Gross Salary : %.2f "%gross_salary) print("Net Salary : %.2f"%net_salary)
