Program :

8. Write an interface called Exam with a method Pass(int mark) that returns a Boolean. Write another interface called Classify with a method Division(int average) which returns a string. Write a class called Result which implements both Exam and Classify. The pass method should return true if the marks is greater than or equal to 35 else false. The division method must return “First” when the parameter average is 60 or more, “second” when average is 50 or more but below 60, “no division” when average is less than 50.


Program :

import java.util.Scanner; interface Exam { boolean pass(int marks); } interface Classify { String division(int average); } class Result implements Exam,Classify { public boolean pass(int marks) { if(marks>=35) return true; return false; } public String division(int average) { if(average>=60) { return "first"; } else if(average>=50 && average<60) { return "Second"; } else { return "No Division"; } } } class Main { public static void main(String args[]) { Scanner sc=new Scanner(System.in); Result r1=new Result(); int marks; int avg=0,sum=0; System.out.println("Enter marks :"); marks=sc.nextInt(); System.out.println(r1.pass(marks)); System.out.println(r1.division(marks)); } }

9. Create class calculation with an abstract method area( ). Create Rectangle and Triangle subclasses of calculation and find area of rectangle and triangle.


Program :

abstract class Calculation { abstract void area(); } class Rect extends Calculation { int length,width; Rect(int l,int w) { length=l; width=w; } public void area() { System.out.println("Area of rectangle is :"+(length*width)); } } class Triangle extends Calculation { int base,height; Triangle(int b,int h) { base=b; height=h; } public void area() { System.out.println("Area of Triangle is :"+(0.5*base*height)); } } class Main { public static void main(String args[]) { Rect r1=new Rect(10,20); Triangle t1=new Triangle(10,20); r1.area(); t1.area(); } }
